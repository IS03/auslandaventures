# Ausland Aventuras - Project Rule

Every UI for this project must feel custom-designed, polished and production-ready.

## Product direction

Ausland Aventuras is a Córdoba-based travel agency / adventure tourism brand.

The product must feel:
- Fresh.
- Adventurous.
- Warm.
- Trustworthy.
- Human.
- Modern.
- Easy to use.

Avoid:
- Generic AI-looking pages.
- Empty white card layouts.
- Random purple gradients.
- Cold corporate design.
- Overly luxurious aesthetics.
- Dense text blocks.

## Visual direction

Use:
- Large destination photography.
- Warm travel copy.
- Soft organic shapes.
- Subtle map/route details.
- Compass, palm, mountain, flight, bus or Córdoba-inspired details.
- Cohesive colors inspired by sky blue, turquoise, warm yellow/orange and deep navy.
- Microinteractions.
- Responsive mobile-first layouts.

## Functional direction

The website must prioritize:
- Destination discovery.
- Clear categories.
- WhatsApp consultation.
- Simple trip information.
- Trust and social proof.

## Reference interpretation

Buenas Vibras:
Use as inspiration for functional details: filters, categories, trip cards, price labels, community, footer.

Lunagui Viajes:
Use as inspiration for the hero/home feeling: large travel image, emotional copy, clear CTA.

Do not copy either website literally.
