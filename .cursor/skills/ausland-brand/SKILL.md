# Ausland Aventuras — Brand

Read before UI work: `docs/brief-ausland.md`, `docs/referencias-visuales.md`, `docs/assets.md`, `src/data/contact.ts`.

## Brand

- **Name:** Ausland Aventuras
- **Tagline:** La aventura de conocer un nuevo destino.
- **Location:** Córdoba, Argentina
- **Tone:** Warm, close, adventurous, simple, trustworthy — not corporate or ultra-luxury.

## Visual

- **Palette:** Sky blue, turquoise, warm yellow/orange, deep navy.
- **Logos:** `public/brand/` — prefer `logo-horizontal.png` (header), `logo-blanco.png` (on photos), `logo-cuadrado.png` (icon).
- **Motifs:** Subtle map/route lines, compass, mountains, palms, plane, Córdoba arch — never clutter.
- **Photos:** Large destination imagery; avoid generic AI purple gradients and Inter-only layouts.

Pair with the `frontend-design` skill for execution quality.

## Product

- **Primary CTA:** WhatsApp (`src/data/contact.ts` — use primary number for global CTAs).
- **Categories:** Nacionales · Internacionales · Regionales / Full Day
- **Data:** `src/data/destinations.ts` for destinations and per-trip WhatsApp messages.

## References (inspiration only)

- **Buenas Vibras:** filters, cards, prices, social proof, footer.
- **Lunagui Viajes:** emotional hero, large image, clear CTA.

Do not copy either site literally.

## Assets

- Web logos → `public/brand/`
- Destination photos → `public/destinos/` (paths in `destinations.ts`)
- Vertical social flyers → `public/marketing/` (not raw heroes without crop/optimize)
